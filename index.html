Syntax

  • One frequency component per line
  • Use t as the variable, p for π , and i for √-1
  • Constant expressions (expr) do not contain the variable t, vice-versa for variable expressions (var_expr)
  • Expressions can contain the operators [+, -, *, /, sin(), cos(), exp()] and (parantheses).
  • If a line contains only constant expressions, it may be written in any valid format. (e.g: (1+i)/(1-i) - 4*sin(p/2))
  • If a line contains a variable expression, it must be of the form expr*expr*...*func(var_expr). func() can be sin()/cos()/exp(). var_expr cannot contain sin()/cos()/exp(). func(var_expr) must be periodic.
  • Example of a variable expression: 2/(3i + 4) * exp(-3i) * cos (0) * sin (t - p/2)
  • i, p, and t can be chained with numbers without the need for a * sign. (e.g. 2ipt = 2*i*π*t , 3t/4i = (3*t)/(4*i))
  • Use // to comment out a line.
